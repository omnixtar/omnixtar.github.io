## GRATIS代币
- **以用户的数字产权收费, 换取使用 AI词元算力 的许可**

---

- [**GRATIS代币**](../fcn/a1) (Grand Unified AI Token Trading Platform 统一AI代币交易平台): 构成激励层，用微支付奖励每一个贡献算力的参与者。
  - 拉丁语 GRATIS "免费": 隐喻 GRATIS代币**不需要用户付费**获得, 而是:
    - 通过 **Omnihash 全杂数** 实现 **以用户的数字产权收费, 换取使用 AI词元算力 的许可**。 
    - **代币绑定**：`ai_token.token_id` 与 Omnihash 关联，确保代币唯一性与所有权。

### DJSON 的验证与使用

1. **身份验证**：`current_user_ID` 和 `prev_msg_owner` 通过 Omnihash 公钥验证交易双方身份。
2. **合约验证**：`doc_hash` 指向链上存储的代币发行合约或市场协议。
3. **代币绑定**：`ai_token.token_id` 与 Omnihash 关联，确保代币唯一性与所有权。
4. **I2P 可访问性**：`i2p_address` 指定代币对应的 AI 计算节点的 I2P 网络地址。
5. **过期机制**：`terms.end_date` 定义代币失效时间，支持自动过期管理。

此 DJSON 可直接用于去中心化交易平台、钱包或智能合约触发器，实现 TMX 到 AI 算力代币的完整交易闭环。

### 字段详解

#### 顶层字段（DJSON 标准格式）

| 索引 | 字段 | 值 | 说明 |
|------|------|-----|------|
| `[0]` | 时间戳 | `2026-07-14T15:20:00.000+0800` | 交易发起时间 |
| `[1]` | 操作类型 | `tmx_buy_ai_token` | 自定义动作：用 TMX 购买 AI 代币 |
| `[2]` | 当前用户 ID | `CXAGcRKevA==` | 买方 Omnihash 公钥哈希 |
| `[3]` | 前一所有者 | `HymWBzfj9A==` | 卖方/发行方 Omnihash |
| `[4]` | 文档哈希 | `DgV6_qnujw==` | 代币发行合约或市场协议的 Omnihash |
| `[5]` | 交易详情 | 嵌套对象 | 具体交易参数 |

---

#### 交易详情对象（`[5]`）

| 字段 | 值 | 说明 |
|------|-----|------|
| `pair` | `TMX/GRATIS` | 交易对 |
| `side` | `buy` | 买入方向 |
| `price` | `0.0000000125` | 每单位 GRATIS 的 TMX 价格（按 10 亿枚总价 12.5 TMX 计算） |
| `quantity` | `1000000000` | 购买数量（10 亿枚） |
| `total` | `12.50` | 总价值（TMX 计价） |
| `fee` | `0.025` | 手续费（0.2%） |
| `fee_token` | `TMX` | 手续费币种 |
| `order_type` | `limit` | 限价单 |
| `tx_hash` | `0x7a8b...` | 链上交易哈希 |
| `expiry` | `2026-07-14T16:00:00.000+0800` | 订单有效期（1 小时） |

---

#### AI Token 详情对象（`ai_token`）

| 字段 | 值 | 说明 |
|------|-----|------|
| `token_id` | `GRATIS-omnihash-abc123` | 代币唯一标识符（绑定 Omnihash） |
| `name` | `GRATIS AI Compute Token` | 代币名称 |
| `symbol` | `GRATIS` | 代币符号 |
| `total_supply` | `1000000000` | 总供应量（10 亿枚） |

**网络信息（`network`）**

| 字段 | 值 | 说明 |
|------|-----|------|
| `i2p_address` | `xyz.....b32.i2p` | I2P 网络地址（Base32 格式） |
| `protocol` | `I2P` | 底层网络协议 |

**AI 模型信息（`ai_model`）**

| 字段 | 值 | 说明 |
|------|-----|------|
| `mode` | `ggml6B` | AI 模型运行模式 |
| `type` | `GGML quantized model` | 模型类型（GGML 量化） |
| `version` | `6B-parameter` | 模型参数规模（60 亿参数） |
| `compatibility` | `["llama.cpp", "whisper.cpp", "stable-diffusion.cpp"]` | 兼容的后端引擎 |

**使用条款（`terms`）**

| 字段 | 值 | 说明 |
|------|-----|------|
| `duration` | `1 year` | 有效期（1 年） |
| `start_date` | `2026-07-15T00:00:00.000+0800` | 生效日期 |
| `end_date` | `2027-07-15T00:00:00.000+0800` | 到期日期 |
| `usage_rights` | `unlimited inference for contributed compute` | 使用权说明（贡献算力者可无限推理） |
